# Personal Project Portfolio 🚀
Custom built personal portfolio website designed to showcase my projects. 
## 🛠️ Tech Stack
+ Frontend: HTML, CSS
+ Interactivity: JavaScript
+ Typography: Google Fonts (Poppins)
## 🌟 Key Features
+ Design: Optimized for all screen sized from mobile to desktop size
+ Amination: Custom animation in CSS and JavaScript visualising Sydney landmarks and simulation of Turing Machine
+ Contact Form: Integrated with the Fetch API to handle user messages without reloding the page
